Brain Gains is a flashcard app that uses the basic methods of the Leitner System. You can use it to write flash cards in question and answer form. Then quiz yourself on the topic. By practicing recall with spaced repetition, we ensure we retain the information better.

The Leitner system is a method to schedule how often you might test recalling the answer to a flash card question.

BrainGains helps keep all of this simple and organized for you.


Instructions


1. Take a Course
2. Write Flashcards as Notes
3. Quiz Daily
4. Repeat until you're not offered any questions.

How it Works


Session 1:
To start, all questions are in Box 1.
Quiz on those questions, Correct questions move up one box, incorrect questions go back to Box 1.
You will rate how you did on the question, choosing correct or incorrect.

Session 2:
The next day you would quiz, using Box 1 and Box 2 questions.
Correct questions move up one box, incorrect questions go back to Box 1.

Session 3:
Day 3, quiz Box 1, Box 2, and Box 3 together.
Correct questions move up one box, incorrect questions go back to Box 1.

Leitner suggests the following spacing:

  • Box 1: Review daily.
  • Box 2: Review every 2 days.
  • Box 3: Review every 4 days.
  • Box 4: Review every 8 days.
  • Box 5: Review every 16 days.

This process repeats over time until you've graduated the question, it moves into Box 6, and you will no longer be asked it.
